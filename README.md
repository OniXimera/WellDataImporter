Инструкция по запуску
Для запуска проекта необходим установленный .NET 8 SDK.

1. Склонируйте репозиторий:

git clone [ссылка]

2. Перейдите в папку проекта:

cd WellDataImporter

3. Запустите приложение:

dotnet run --project WellDataImporter/WellDataImporter.csproj

---------------------------
Реализованные особенности (Extra Features)
Архитектура SOLID & DI: Полное разделение ответственности. Использование паттерна Dependency Injection (внедрение зависимостей) (чуточку) через корень композиции (App.xaml.cs). Логика парсинга, расчетов и экспорта полностью изолирована.

Unit-тестирование: Проект покрыт тестами с использованием xUnit и Moq. Протестированы:

1. Граничные случаи парсинга CSV (мусорные данные, пустые строки).

2. Сложная математика калькулятора (взвешенная пористость, доминирующая порода).

3. Логика пересечения интервалов (Overlaps).

4. Экспорт в JSON через Mock-файловую систему.


Валидация данных: Реализована проверка на отрицательные глубины, некорректную пористость и пересекающиеся интервалы внутри скважины.

Локализация: Все системные сообщения и ошибки вынесены в файлы ресурсов (.resx), что позволяет легко добавить поддержку других языков.

MVVM Toolkit: Использование современного стека CommunityToolkit.Mvvm для реактивного UI и команд.

Абстракция ФС: Создана прослойка IFileSystem, позволяющая тестировать логику работы с файлами без обращения к реальному диску.
